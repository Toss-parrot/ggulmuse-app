# 껄무새 PRD — 앱인토스 MVP

---

## 한 줄 요약

**"그때 샀더라면?" — 과거 투자를 가상 체험하는 시간 여행 미니앱**

---

## Problem

- "그때 삼전 샀으면…", "비트코인 1만원일 때…" — 누구나 하는 후회
- 후회는 감정으로만 남고, **구체적 숫자로 체감한 적이 없음**
- 기존 수익률 계산기는 숫자만 보여줘서 재미가 없음

## Solution

과거 시점에 투자했다면 **지금 얼마가 됐을지**, 타임라인을 따라가며 체험해요. 중간에 급등/폭락이 오면 "팔겠습니까?" 선택지가 나와서 **내가 진짜 버텼을지**까지 시험해요.

## Target User

| 세그먼트 | 설명 |
| --- | --- |
| **1차** | 주식/코인 해본 2030 |
| **2차** | 안 해봤지만 관심 있는 사람 |
| **바이럴** | SNS에서 껄무새 카드 보고 유입 |

## Core Flow

```
[종목 선택] → [시점 선택] → [금액 입력] → [시간 여행] → [결과 + 공유]
```

| # | 화면 | 사용자 액션 | 핵심 UX |
| --- | --- | --- | --- |
| 1 | 홈 | 인기 종목 탭 or 검색 | "요즘 다들 후회하는 종목" 큐레이션 |
| 2 | 시점 선택 | 캘린더에서 날짜 선택 | "5년 전 오늘" "비트코인 1만원일 때" 숏컷 |
| 3 | 금액 입력 | "100만원 투자했다면?" | 슬라이더 + 직접 입력 |
| 4 | 시간 여행 | 타임라인 스크롤 | **핵심 경험** (아래 상세) |
| 5 | 결과 카드 | 공유 탭 | 껄무새 카드 생성 + 공유 |

### 시간 여행 (핵심 경험)

타임라인을 스크롤하면 **시간이 흘러가요**:

```
─────────────────────────────────────────
 2021.01  ██░░░░░░░░  +12%   100만→112만
          📰 바이든 취임, 반도체 수급 이슈

 2021.06  █████░░░░░  +45%   100만→145만
          🔥 최고점 근처! 지금 팔겠습니까?
          [계속 보유]  [여기서 매도]

 2021.12  ███░░░░░░░  +28%   100만→128만
          📉 금리 인상 시그널

 2022.06  █░░░░░░░░░  -15%   100만→85만
          💀 약세장 진입. 버티겠습니까?
          [존버]  [손절]
─────────────────────────────────────────
```

**선택에 따라 결과가 갈려요** — 단순 계산기가 아니라 **내 심리까지 시험하는 게임**

## 기술 구성

| 구성 | 기술 |
| --- | --- |
| 프론트 | 앱인토스 웹앱 (React) |
| 백엔드 | 서버리스 or 경량 API 서버 |
| 주가 데이터 | 한국거래소 공개 API / Yahoo Finance |
| 이벤트 데이터 | 주요 뉴스 사전 큐레이션 (날짜별 매핑) |
| 카드 생성 | Canvas API → 이미지 export |
| 공유 | 카카오톡 / 인스타스토리 / 링크 복사 |

### 앱인토스 장점

- **토스 앱 안에서 바로 실행** — 별도 설치 없음
- **토스 유저 풀에 노출** — 앱인토스 탭에서 자연 유입
- **가벼운 진입** — 웹앱이라 로딩 빠르고 부담 없음

| 카드 생성율 | 40% |

| 재방문 D7 | 20% |

## 바이럴 루프

```
A가 시뮬레이션 완료
  → 껄무새 카드 🦜 "+340%였는데..."
    → 카톡/인스타 공유
      → B가 "나도 해볼래" → 토스 앱 → 앱인토스
        → B도 카드 공유 → 반복
```

핵심: **카드에 박힌 수익률 숫자가 곧 광고**. 별도 마케팅 비용 없이 퍼짐.

## Risk

| 리스크 | 대응 |
| --- | --- |
| 투자 조장 논란 | "교육/엔터테인먼트" 포지셔닝 + 면책 문구 상시 노출 |
| 과거 수익률 오해 | "과거 성과 ≠ 미래 보장" 고정 표시 |
| 데이터 정확도 | 한국거래소 공식 데이터 + 수정주가 반영 |
| 하락장만 안 보여줌 | "하락장 체험" 모드 기본 제공 (리먼, 코로나 등) |
